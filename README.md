### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GloChatv1/glochat1
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the server
   ```sh
   node server.js
   ```
5. Visit the site
   ````sh
   http://localhost:3000
   ```
